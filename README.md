## Installation

    git clone git://github.com/ianp/dotvim.git ~/.vim

## Create Symlinks

    ln -s ~/.vim/vimrc ~/.vimrc
    ln -s ~/.vim/gvimrc ~/.gvimrc

## Fetch Submodules

Switch to the `~/.vim` directory, and fetch submodules:

    cd ~/.vim
    git submodule init
    git submodule update
